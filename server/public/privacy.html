Privacy Policy

Last updated: June 2026

Overview

The PhoneBurner Dial Session Companion Chrome Extension helps PhoneBurner customers launch PhoneBurner dial sessions from supported CRM platforms and follow calls in real time. The extension runs only when a user initiates an action.

Information We Collect

The extension may access the following data only when the user initiates an action (for example, launching a dial session):

  1. Personally identifiable information from your CRM
    • Contact names
    • Phone numbers
    • Email addresses

    This information is read from your CRM to create and follow PhoneBurner dial sessions. The extension does not independently create, enrich, or sell this data.

  2. Authentication information

    A PhoneBurner Personal Access Token (PAT), and (for Level 3 CRMs) OAuth tokens that authorize the extension to read your CRM data.

    How and where these tokens are stored: When you save your PAT or connect a CRM, the credential is transmitted over HTTPS to PhoneBurner-managed servers and stored with strict owner-only file permissions (mode 0600). Your browser does not keep a copy of these tokens. The browser stores only an anonymous lookup key (a random UUID) that we use to retrieve the token server-side at the moment a dial session is created. A compromised browser profile or another extension cannot read your PhoneBurner credentials.
  3. Website content

    CRM page content necessary to detect selected records or extract contact details. The extension reads only the minimum content required to perform the requested action, and only at the moment you initiate that action.

How We Use Information

Collected information is used only to:

  • Launch PhoneBurner dial sessions
  • Display the currently-called contact during an active dial session
  • Provide real-time call follow-along functionality inside supported CRMs
  • Log call activities back to your CRM (where supported by the integration — HubSpot, Close, Apollo)

Data Storage

  • Tokens (PAT and OAuth): stored on PhoneBurner-managed servers with owner-only file permissions. Not stored in your browser.
  • Browser-local data: the extension keeps an anonymous client identifier (UUID) and your user preferences (e.g., preferred phone field) in Chrome's local storage. No personally identifiable contact data is persisted in browser storage.
  • Session data: active dial session state is temporarily processed by PhoneBurner servers to support real-time call tracking. Sessions are cleaned up after they end.

What We Do Not Do

  • We do not sell, rent, or trade user data
  • We do not track browsing history
  • We do not monitor keystrokes, clicks, or user activity
  • We do not collect health, financial, or location data
  • We do not use data for advertising or third-party analytics

Data Sharing

Data is shared only with PhoneBurner services (and, for Level 3 CRMs, with the CRM's API on your behalf, using the OAuth tokens you authorized) as required to create and manage dial sessions. No data is shared with third parties outside of this purpose.

User Control

  • The extension operates only when you interact with it.
  • You can disconnect your PhoneBurner account or any connected CRM at any time via the extension's Settings tab. Disconnecting deletes the corresponding server-side token immediately.
  • Removing the extension from Chrome stops all browser-side access. To also delete server-side tokens, click Disconnect in the extension Settings before uninstalling.

Changes to This Policy

This policy may be updated to reflect changes in functionality or compliance requirements. Updates will be posted on this page.
