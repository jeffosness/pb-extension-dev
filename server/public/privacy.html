Privacy Policy

Last updated: January 2026

Overview

The PhoneBurner Dial Session Companion Chrome Extension helps users launch PhoneBurner dial sessions from supported CRM platforms and follow calls in real time. The extension runs only when a user initiates an action.

Information We Collect

The extension may access the following data only when the user initiates an action (for example, launching a dial session):

  1. Personally Identifiable Information
    • Contact names
    • Phone numbers
    • Email addresses

    This information is read from the user's CRM to create and follow PhoneBurner dial sessions. The extension does not independently create, enrich, or sell this data.

  2. Authentication Information

    A PhoneBurner Personal Access Token (PAT). This token is used to authenticate with PhoneBurner APIs and is stored locally in the user's browser.

  3. Website Content

    CRM page content necessary to detect selected records or extract contact details. The extension reads only the minimum content required to perform the requested action.

How We Use Information

Collected information is used only to:

  • Launch PhoneBurner dial sessions
  • Display the currently called contact during an active dial session
  • Provide real-time call follow-along functionality inside supported CRMs

Data Storage

Authentication tokens and user preferences are stored locally using Chrome's storage APIs. Session data may be temporarily processed by PhoneBurner servers to support real-time call tracking. The extension does not maintain long-term user data storage.

What We Do Not Do

  • We do not sell, rent, or trade user data
  • We do not track browsing history
  • We do not monitor keystrokes, clicks, or user activity
  • We do not collect health, financial, or location data
  • We do not use data for advertising or analytics

Data Sharing

Data is shared only with PhoneBurner services as required to create and manage dial sessions. No data is shared with third parties outside of this purpose.

User Control

  • The extension operates only when the user interacts with it.
  • Users can disconnect their PhoneBurner account at any time via the extension UI.
  • Removing the extension stops all data access from the browser.

Changes to This Policy

This policy may be updated to reflect changes in functionality or compliance requirements. Updates will be posted on this page.
