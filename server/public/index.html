How it works

  • Install the extension and connect your PhoneBurner Personal Access Token (PAT).
  • From any supported CRM page click the extension and scan the list or selection.
  • The extension creates a PhoneBurner dial session and opens it in a popup — start dialing immediately.

Integration Levels

We support three integration levels so the extension works well across different CRMs and page layouts.

Level 1 — Generic

No CRM-specific code. We attempt to parse the visible page to extract phone numbers. In our testing, Level 1 succeeds ~50% of the time depending on CRM layout.

Level 2 — Optimized Scraping

We’ve analyzed the CRM layout and tuned extraction rules to improve success when parsing table/list views.

Level 3 — API Integration

Deep integration using the CRM API to reliably obtain records and metadata. HubSpot is Level 3 and supports calling contacts, deals, and companies.

Who this is for

Sales teams who want to dramatically increase outbound call volume without changing CRMs. If you use PhoneBurner and want to dial from your CRM lists, this extension connects the flows.
