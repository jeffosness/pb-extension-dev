PhoneBurner Dial Session Companion

Launch and follow PhoneBurner dial sessions directly from supported CRMs

What this extension does

This Chrome extension helps PhoneBurner customers scan CRM pages, create dial sessions, and follow live calls with an on-screen "Follow" widget. It supports optimized HubSpot integration (Level 3) plus generic scraping for other CRMs.

How it works

  • Click the extension and scan a CRM page to collect selected contacts.
  • The extension posts the scanned contacts to the backend API which returns a `session_token` and `launch_url`.
  • The background service worker opens the PhoneBurner dial session in a popup and can restore a lightweight follow widget in the CRM tab.

Run locally

Run the backend for local testing from the repository root:

php -S 127.0.0.1:8000 -t server/public
